Design tokens for the Premium AI SaaS System.

Files:
- `design-system.json` — central color, typography, spacing, and radius tokens.

Usage:
1. Ensure `tailwind.config.js` is present at `fastapiapp/frontend/tailwind.config.js`.
2. Install dependencies if you haven't: `npm install` (or `pnpm`/`yarn`).
3. Build or run your frontend — the Tailwind config will load the tokens automatically.
